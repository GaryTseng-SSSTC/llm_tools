一頁秒懂 MCP

Model Context Protocol

把「讓 AI 做事」變得像插 USB 一樣簡單——一條協定，連接萬物

MCP 是什麼？

Anthropic 提出的開放標準

為 AI 模型建立統一的通信協定

為 LLM 建立「USB-C」式通用介面

標準化的連接方式，簡化整合流程

透過 JSON-RPC 安全呼叫外部資料與工具

安全且高效的資料交換機制

為什麼要用 MCP？

一次接入→多源共用

免除 N×M 客製 API 整合，大幅降低開發複雜度

即時存取資料

不需預先索引，降低延遲與運算成本

內建權限控制

強化企業安全合規，內建審計功能

最快看到的價值

IDE 助理

讀檔、查 Git、寫測試立刻回覆，提升開發效率

企業查詢

自然語言拉 SQL / ERP 報表，簡化資料分析流程

自動流程

AI 直接發信、建工單、更新 CRM，實現業務自動化

MCP 的應用場景

開發者工具

Zed、Cursor 透過 Filesystem/Git MCP 伺服器即時分析整個專案

商業自動化

AI 連接 Slack、Gmail、JIRA，完成任務指派與進度回報

供應鏈 / 財務

多代理協作即時抓 KPI、偵測庫存或詐欺

IoT 與智慧家居

AI 透過 MCP 控制感測器、執行排程
